Encrypted Filesystem
====================

### Usage:
  1. `$ make pa5-encfs`
  2. `$ ./pa5-encfs <DIR_TO_MIRROR> <MOUNTPOINT>`

### Makefile Commands:
  * ```$ make pa5-encfs```
  * ```$ make mount```
   * Creates a test fuse mount with mnt/ and mir/ directories
  * ```$ make unmount```
   * Unmounts the test fuse mount
  * ```$ make clean```
